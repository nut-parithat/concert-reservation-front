# Next.js + Tailwind CSS Starter

This project was bootstrapped with [Create Next App](https://nextjs.org/docs/api-reference/create-next-app) and includes Tailwind CSS, TypeScript, ESLint, App Router, and a `src` directory structure.

## Getting Started


First, install dependencies:

```bash
pnpm install
```

Then, run the development server:

```bash
pnpm run dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.

## Features
- Next.js (App Router)
- Tailwind CSS
- TypeScript
- ESLint
- `src/` directory structure

## Customization
You can start editing the page by modifying `src/app/page.tsx`. The page auto-updates as you edit the file.
